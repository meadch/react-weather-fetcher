###App Blueprint:

+ MainComponent

    + WeatherForm Component
  + Prompt Component
    + WeatherForm Component
  + Forecast Component
    + Title information
    + WeatherSummary Component
  + Details Component
  + FooterComponent

####Other components/services needed:
+ Router (react-router)
+ API calls
